## Objetivo do Projeto
O projeto visa realizar uma Análise Exploratória de Dados (EDA) crítica sobre as ocorrências de acidentes de trânsito registradas pela Polícia Rodoviária Federal (PRF) nas BRs que cortam o estado do Ceará, avaliando a correlação de fatores ambientais (clima, iluminação) e comportamentais com a frequência dos eventos.

## 📁 Estrutura do Repositório
* `/dados`: Contém os microdados brutos em formato `.csv` coletados do portal do governo federal.
* `/notebooks`: Scripts em Python (`.ipynb`) com os códigos de limpeza, tratamento e visualizações.
* `/relatorio`: Arquivos de documentação e relatório final estruturados em LaTeX.

## Tecnologias e Bibliotecas Utilizadas
* Python 3
* Pandas (Manipulação e tratamento dos dados)
* Matplotlib & Seaborn (Geração de gráficos estatísticos)
* LaTeX (Estruturação do relatório científico)
